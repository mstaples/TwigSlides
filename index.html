Installation & Setup

Option #1: Install Symfony Standard Edition

(If you prefer a custom Symfony2 install, remember to include Twig in your composer.json)
command
$ php composer.phar create-project symfony/framework-standard-edition /path/to/webroot/MyProject '2.5.*'

https://getcomposer.org/

Installation & Setup

Option #1: Install Symfony Standard Edition

Render a template.
                namespace MyTeam\Game\Controller;

                class DefaultController extends Controller
                {
                    public function characterSheetAction()
                    {
                        return $this->render('MyTeamGameBundle:Default:character_sheet.html.twig');
                    }
                }

    Layouts and Blocks

    Add Blocks to the Layout

  • Blocks designate areas that may be replaced by child templates.
  • Block content in the layout will be the default.
    •                 <html>
                    {% block head %}
                        <head>
                            
                            
                                {% block title %}Game Title{% endblock %}
                            
                        </head>
                    {% endblock %}

                    <body>
                        
    {% block content %}{% endblock %}
    


                        
    
                            {% block footer %}
                                © Copyright 2015 by MyGamesCom.
                            {% endblock %}
                        
    
                    </body>
                </html>
    layout.html(.twig)

    Variables: Strings Filters 

                        {% block content %}
                        
    
                            {{ name|replace({ 'Sir': 'Lady', 'Knight': 'Ranger' }) }}
                        
    
                        
    
                            Level {{ level|number_format }} {{ adventure_class|title }}
                        
    
                    {% endblock %}
    character_sheet.html(.twig)

    Lady Ranger

    Level 1,000 Adventurer

    render
  • The title filter shown applied to adventure_class.
  • Applying upper or lower filters would print: ADVENTURER or adventurer.
  • The replace filter shown applied to name.
  • The number_format filter shown applied to level.

    • Variables: Other Filters

    More Filter Examples

  • Absolute: {{ -23|abs }} prints 23.
  • Round: {{ 3.141592|round }} prints 3.
  • Round: {{ 3.141592|round(2,'ceil') }} prints 3.15.
  • First: {{ 12345|first }} prints 1.
  • Last: {{ { 'a', 'b', 'c' }|last }} prints c.

    • http://twig.sensiolabs.org/doc/filters/index.html

    Logic 

                        {% block content %}
                        {% if 'Health' in attributes|keys and attributes.Health > 0 %}
                            
    
                                {% set has_attributes = attributes|length %}
                                {% for attribute,value in attributes %}
                                    
    {{ attribute }}: {{ value|number_format }}
    
                                {% endfor %}
                                
    {{ name }} has {{ has_attributes }} attributes.
    
                            
    
                        {% else %}
                            
    
                                
    {{ name }} is Dead!
    
                            
    
                        {% endif %}
                    {% endblock %}

    Include

  • A template may include other templates to assemble a page in pieces.
  • In Symfony2 projects, include the full bundle name in your include.
    •                     {% if 'Health' in attributes|keys and attributes.Health > 0 %}

                        {% include 'attributes.html' with  { 'attributes': attributes } %}

                    {% else %}

                        {% include 'dead_character.html' %}

                    {% endif %}
